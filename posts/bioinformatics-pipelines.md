Our repository for bioinformatics pipelines contains a comprehensive set of
workflows developed in Nextflow for common practices in bioinformatics with a
focus on microbiome data analysis. This includes reference-based and
assembly-based metagenomics, metatranscriptomics, RNA-seq, strain-level
metagenomics and so on. All processes are containerized and can be run on any
system with Docker or Apptainer installed. Check out the GitHub repository for
this project: [Olm Lab's bioinformatics pipelines](https://github.com/OlmLab/bioinformatics_pipelines).
